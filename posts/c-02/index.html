1. 연산자#

C 언어에서는 기본적으로 연산자를 사용하여 계산합니다.

구분연산자
대입 연산자=
산술 연산자+, -, *, /, &, ++, –
관계 연산자<, >, <=, >=, ==, !=
논리 연산자&&, ||, !
할당 연산자+=, -=, *=, /=, %=, 등 …
삼항 연산자?
비트 연산자&, |, ~, ^, «, »

대입 연산자#

변수에 값을 넣을 때 사용했던 = 연산자를 대입 연산자라고 합니다.

a = 1;

이라고 했을때, 1을 a라는 변수에 넣습니다. 이와 같이 대입 연산자는 오른쪽의 값을 왼쪽에 대입하는 역할을 합니다.

산술 연산자#

우리가 실제 산수에 사용하는 연산자와 유사합니다. 좀 더 자세히 알아보면

연산자기능
+더하기
-빼기
*곱하기
/나누기
%나머지
++1을 증가시킴
–1을 감소시킴

이와 같습니다. 예를 들어보면

a = 1 + 1;  //1과 1을 더해 a에 넣는다.

a = 3 - 1;  //3에서 1을 빼서 a에 넣는다.

a = 2 * 3;  //2와 3을 곱한 후 a에 넣는다.

a = 4 / 2;  //4를 2로 나눈 몫을 a에 넣는다.

a = 5 % 2;  //5를 2로 나눈 나머지를 a에 넣는다.

다음과 같습니다.

a++, a--

++a, --a

이런 형태를 따로 증감 연산자 라고도 하는데, 변수에 ‘++‘을 붙이게 되면 ‘1’을 증가시키고, ‘–‘를 붙이면 ‘1’을 감소 시킵니다. 이 ‘증감 연산자’가 변수 앞에 붙을때와 뒤에 붙을때의 차이점이 존재하는데, 다음을 봅시다.

int a = 1, b = 0;

일때

b = a++;    // b = 1, a = 2 -> 뒤에 증감 연산자가 붙은 경우
b = a--;    // b = 1, a = 0 -> 뒤에 증감 연산자가 붙은 경우

b = ++a;    // b = 2, a = 0 -> 앞에 증감 연산자가 붙은 경우
b = --a;    // b = -, a = 0 -> 앞에 증감 연산자가 붙은 경우

앞에 증감 연산자가 붙을 경우 (전위형) : 증감 연산자 계산 -> 대입 뒤에 증감 연산자가 붙을 경우 (후위형) : 대입 -> 증감 연산자 계산

처리속도는 앞에 증감 연산자를 붙이는 전위형이 후위형 보다 약간 빠르기에 둘중 어떤것을 써도 상관없는 상황에서는 전위형을 써주는게 여러모로 좋습니다.

관계 연산자#

두 값의 크기를 비교하는 연산을 합니다.

연산자기능
<왼쪽이 오른쪽보다 작으면 1, 크거나 같으면 0
>왼쪽이 오른쪽보다 크면 1, 작거나 같으면 0
<=왼쪽이 오른쪽보다 작거나 같으면 1, 크면 0
>=왼쪽 오른쪽보다 크거나 같으면 1, 작으면 0
==두 값이 같으면 1, 다르면 0
!=두 값이 다르면 1, 같으면 0

예를 들어 보면

int a = 3, b = 5, c = 0;

일때

c = a < b;  // c = 1

c = a > b;  // c = 0

c = a == b; // c = 0

c = a != b; // c = 1

이와 같습니다. 여기서 값이 1이 되는 경우를 참 (True)라고 하고, 0이 되는 경우를 거짓 (False)라고 합니다.

논리 연산자#

두 값을 해당하는 논리로 연산합니다.

연산자논리기능
&&AND양쪽이 모두 참이면 참
||OR양쪽 중 하나이상 참이면 참
!NOT참이면 거짓, 거짓이면 참

예를 들어

int a = 3, b = 1, c = 0;

일때

c = (a > b) && (b == 1);   // c = 1

c = (a > b) && (b < 1);    // c = 0

c = (a > b) || (b == 1);   // c = 1

c = (a > b) || (b < 1);    // c = 1

c = !(a > b);              // c = 0

c = !(b < 0);              // c = 1

이런식으로 AND 연산은 양쪽 결과가 참일 경우에 1이고, 어느한쪽이라도 거짓이면 0이 됩니다. 반면에 OR 연산은 양쪽 중 하나라도 참이면 1이고, 양쪽 모두 거짓이면 0이 됩니다. NOT 연산은 거짓이라면 1, 참이라면 0이 됩니다. 그런데 다음 예처럼

c = 100 && -10;    // c = 1

c = -20 || -10;    // c = 1

c = !123;          // c = 0

1도 아니고 0도 아닌 값들을 넣어봤습니다. 여기서 알 수 있는건 C언어에서는 0이 아닌 모든 값을 참 (True)으로 처리하고 있다는 겁니다. 1만이 참은 아니라는겁니다.

할당 연산자#

대입연산과 산술연산을 줄여서 같이 쓰는 경우입니다.

연산자기능
+=자신에 오른쪽 값을 더해 넣는다.
-=자신에 오른쪽 값을 빼서 넣는다.
*자신을 오른쪽 값으로 곱해서 넣는다.
/=자신을 오른쪽 값으로 나누어 몫을 넣는다.
%=자신을 오른쪽 값으로 나누어 나머지를 넣는다.

다음의 예를 보면

int a = 10;
 
일때

a += 5;    // a = 15

a -= 5;    // a = 5

a *= 5;    // a = 50

a /= 5;    // a = 2

a %= 5;    // a = 0

간단하게 위의 식은 다음의 식을 줄인 것 입니다.

a = a + 5;

a = a - 5;

a = a * 5;

a= a / 5;

a = a % 5;

삼항 연산자#

지금까지 알아본 연산자들은 2개의 항이 있는 이항 연산자였습니다. 이것만으로도 대부분의 코드를 계산할 수 있지만 3개의 항을 가지는 삼항 연산자를 사용하면 매우 간결하게 코드를 작성할 수 있는 장점이 있습니다.

조건 ? 참일때의 값 : 거짓일때의 값;

이런 형식이며, 예를들어

int a = 0, b = 1, c = 0;
 
일때

c = (a > b) ?  10 : 20;    // c = 20

c = (a == 0) ? 10 : 20;    // c = 10

c = a ? a : b;             // c = 1

이와 같이 ? 왼쪽의 조건이 ’ 참 ‘이면 ’ : ‘의 왼쪽 값을, 거짓이면 ’ : ‘의 오른쪽 값을 반환합니다.

비트 연산자#

컴퓨터의 모든 수는 2진수이고 한자리는 1비트입니다. 비트 연산자는 이 비트 단위로 특정한 연산을 수행합니다.

연산자기능
&비트단위 AND 연산
|비트단위 OR 연산
~비트단위 NOT 연산
^비트단위 XOR 연산
«왼쪽으로 비트 이동
>>오른쪽으로 비트 이동

예를 들어

a = 1010 0101, b = 1001 1001일때

c = a & b;  // c = 1000 0001 -> 양쪽 다 참일때 1

c = a | b;  // c = 1011 1101 -> 양쪽 중 하나라도 참이면 1

c = ~a;     // c = 0101 1010 -> 거짓이면 참, 참이면 거짓으로

c = a^b;    // c = 0011 1100 -> 두 값이 서로 다르면 1, 같으면 0

AND 연산(&) : 양쪽 다 참일떄 1 OR 연산(|) : 양쪽 중 하나라도 참이면 1 NOT 연산(~) : 거짓이면 참, 참이면 거짓 XOR 연산(^) : 두 값이 서로 다르면 1, 같으면 0

a = 0011 1010; 일때

c = a << 1;    // c = 0111 0100 (00111010을 왼쪽으로 한칸씩 이동)

c = a << 2;    // c = 1110 1000 (00111010을 왼쪽으로 두칸씩 이동)


c = a >> 1;    // c = 0001 1101 (00111010을 오른쪽으로 한칸씩 이동)

c = a >> 2;    // c = 0000 1110 (00111010)을 오른쪽으로 두칸씩 이동

이 <<, >>를 쉬프트 연산자라고도 합니다. 2진수의 특성상 왼쪽으로 1비트 이동하면 2를 곱한것과 같은 값이 되고, 왼쪽으로 2비트를 이동하면 2²인 4를 곱한 것과 같은 값이 됩니다. 반대로 오른쪽으로 1비트 이동하면 2를 나눈것과 같은 값이 되고, 오른쪽으로로 2비트를 이동하면 2²인 4를 나눈것과 같은 값이 됩니다. 정리하면

int a = 58;

일때

c = a << 1;   // c = 116 (= a * 2)

c = a << 2;   // c = 232 (= a * 4)


c = a >> 1;   // c = 29 (= a / 2)

c = a >> 2;   // c = 14 (= a / 4)

2. 연산자의 우선 순위#

C 언어에서는 중괄호 {} 나 대괄호 [] 를 계산식에 사용할 수 없고 오로지 소괄호 () 만 사용할 수가 있습니다.

c = 1 + ((1 + 2) * (3 / 1));    // c = 1 + 3 * 3 = 10

이런 식입니다. 그 밖의 연산자까지 모두 정리해보면

연산자표기
단항 연산자++, –, ! 등 …
곱하기, 나누기*, /
더하기, 빼기+, -
쉬프트 연산«, »
관계 연산자>, <, >=, <=, ==, !=
비트 연산자ADN(&), XOR(^), OR(|)
논리 연산자AND(&&), OR(||)
삼항 연산자?
대립 및 할당 연산자=, +=, -=, *=, /= 등 …

이상으로 연산자 설명 정리를 마무리 하겠습니다.
