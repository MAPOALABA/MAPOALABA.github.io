1. 라우터 문제#

  • Router>

  • Router#

  • Router(config)#

  • Router(config-line)#

01. 라우터의 호스트이름을 R1으로 변경하시오#

Router> ena

Router> conf t

Router (config) hostname R1

R1(config)# exit

만약 문제에서 NVRAM에 저장하라고 나온다면 관리자모드에서 명령어를 입력해야함

R1# copy r s

02. 콘솔로 접속했을때 1234로 암호를 설정하시오#

R1# conf t

R1(config)# line console 0 또는 line con 0

R1(config-line)# password 1234

여기까지 암호를 설정하고 만약 로그인시 암호를 묻도록 설정할때 (함호설정 이전에 해도됨)

R1(config-line)# login

저장하기 위해서는 현재 콘솔모드이므로 저장하기 위해서는 두 번의 단계를 나가야 관리자모드로 돌아감

R1(config-line)# exit

R1(config)# exit

R1# copy r s

R1# show run

03. 텔넷에 “1234"로 암호를 설정하고 동시에 접근할 수 있는 개수를 6개로 제한하시오#

Router> ena

Router# conf t

Router(config)# line vty 0 5

만약 개수가 안나온다면 default는 line vty 0 4

Router(config-line)# password 1234

Router(config-line)# login

Router(config-line)# end

Router# copy r s

Router# show run

04. 콘솔모드에서 “hello"로 암호를 해독할 수 없도록 설정하시오#

Router> ena

Router# conf t

Router(config)# service password-encryption

Router(config)# line con 0

Router(config-line)# password hello

Router(config-line)# end

Router# copy rs

Router# show run

<show run 결과>

line con 0

password 7 082949420516

05. 관리자모드로 관리자가 로그인할 때 암호를 묻도록 설정하시오(암호는 cisco)#

Router> ena

Router# conf t

Router(config)# enable paswword cisco

Router(config)# exit

Router# copy rs

Router# show run

06. 관리자모드로 로그인할 때 패스워드 입력은 같지만 암호화되도록 하시오#

Router> ena

Router# conf t

Router(config)# enable secret cisco

Router(config)# exit

Router# copy r s

Router# show run

만약 이미 enable password를 통해 암호를 설정했다면 중복되지 않은 암호를 설정해줘야하며, 이전에 설정된 암호의 기록은 show run에서 삭제되지 않고 보여지며, 암호화된 비밀번호는 암호화되서 보여지고 접속할때 입력해야하는 암호는 암호화된 두번째 암호로 로그인 할 수 있다.

07. 라우터에 접속했을 때 사용자에게 welcome 이라는 배너메세지가 나오게 설정하시오#

Router> ena

Router# conf t

Router(config)# banner motd ‘welcome’

Router(config)# exit

Router# copy r s

Router# show run

08. Consol포트로 접속할 때, 암호를 묻는 Console 패스워드를 설정하고, 현재 상태를 NVRAM에 저장하시오(단, Console 패스워드는 “1234”)#

Router> ena

Router# conf t

Router(config)# line con 0

Router(config-line)# password 1234

Router(config-line)# login

Router(config)# exit

Router# copy r s

09. Enable 패스워드를 “1234"로 설정하고 현재 상태를 NVRAM에 저장하시오#

Router> ena

Router# conf t

Router(config)# line con 0

Router(config-line)# password 1234

Router(config-line)# login

Router(config-line)# end

Router# copy r s

10. Hostname과 Secret 패스워드를 Router1에 설정하고, 현재 상태를 NVRAM에 저장하시오#

Router> ena

Router# conf t

Router(config)# hostname R1

R1(config)# enable secret 1234

R1(config)# exit

R1# copy r s

11. Router에 텔넷포트로 원격접속할 때 암호를 요구하도록 패스워드를 설정하고 NVRAM에 저장하시오#

Router> ena

Router# conf t

Router(config)# line vty 0 4

Router(config-line)# password 1234

Router(config-line)# login

Router(config-line)# end

Router# copy r s

2. 라우터 인터페이스 설정#

192.168.12.2/24 라고 문제가 출제되었을때 /뒤의 24는 32 - 8 이기에 네트워크 부분이 3옥테트이다. 따라서 서브넷 마스크는 class C 서브넷 마스크인 255.255.255.0이 된다.

만약 /뒤의 값이 26bit라면 3옥테트 + 2비트 이므로 255.255.255.11000000 즉 255.255.255.192가 된다.

01. Router의 fastethernet 0/0의 ip를 제시된 내용대로 설정 후 활성화 하시오#

Router> ena

Router# conf t

Router(config)# interface fastethernet 0/0

Router(config-if)# ip address 192.168.3.254 255.255.255.0

Router(config-if)# end

Router# copy r s

만약 fastethernet이 아닌 일반 ethernet의 경우 ethernet 0/0 혹은 e 0/0으로 표기함

serial 인터페이스인 경우 serial 0/0 혹은 s 0/0과 같이 표기함

02. 라우터에서 description을 1234로 설정하고, Ethernet 0번의 IP및 서브넷을 설정하시오 (IP : 210.108.245.1/26, ethernet 1/0, startup-conf에 저장)#

Router> ena

Router# conf t

Router(config)# interface e 1/0

Router(config-if)# description 1234

Routter(config-if)# ip address 210.108.245.1 255.255.255.192

Router(config-if)# no shutdown

Router(config-if)# end

Router# copy r s

Router# ***show interface e 1/0
