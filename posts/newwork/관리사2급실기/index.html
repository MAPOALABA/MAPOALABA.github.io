01. 라우터#

01. 라우팅 테이블을 확인하고 저장하시오#

Router> en

Router# sh ip route

Router# copy r s

02. 주어진 값으로 Router 1의 FastEthernet 0/0 인터페이스를 설정하고, NVRAM에 저장하시오 (IP : 192.168.200.2, 서브넷마스크 : 255.255.255.252)#

Router> en

Router# conf t

Router(config)# int fa0/0

Router(config-if)# ip address 192.168.200.2 255.255.255.252

Router(config-if)# no sh    <!-- 활성화 -->

Router(config-if)# exit

Router(config)# exit

Router# copy r s

Router# sh run

03. Router1의 serial 0/0 인터페이스에 대하여 secondary ip를 설정하고 저장하시오 (IP : 192.168.100.2, 서브넷마스크 : 255.255.255.0)#

Router> en

Router# conf t

Router(config)# int se0/0

Router(config-if)# ip address 192.168.100.2 255.255.255.0 secondary

Router(config-if)# exit

Router(config)# exit

Router# copy r s

Router# sh run

04. Serial 0/0의 대역폭을 2048로 설정하고, NVRAM에 저장하시오#

Router> en

Router# conf t

Router(config)# int se0/0

Router(config-if)# bandwidth 2048

Router(config-if)# exit

Router(config)# exit

Router# copy r s

05. Serial 0/0의 clock rate를 72k로 설정하고, NVRAM에 저장하시오#

Router> en

Router# conf t

Router(config)# int se0/0

Router(config-if)# clock rate 72000

Router(config-if)# exit

Router(config)# exit

Router# copy r s

06. FastEternet 0/0에 대해 주석을 설정하고, NVRAM에 저장하시오 (주석 : ICQA)#

Router> en

Router# conf t

Router(config)# int fa0/0

Router(config-if)# description ICQA

Router(config-if)# exit

Router(config)# exit

Router# copy r s

07. Eternet 0/0에 대해 주석을 설정하고, NVRAM에 저장하시오 (주석 : ICQA)#

Router> en

Router# conf t

Router(config)# int e0/0    // 시험 전 꼭 보기 (Fasteternet이랑 eternet이랑 다름)

Router(config-if)# description ICQA

Router(config-if)# exit

Router(config)# exit

Router# copy r s

08. Default-Gateway를 설정하고, 저장하시오 (IP : 192.168.0.10)#

Router> en

Router# conf t

Router(config)# ip default-gateway 192.168.0.10 // 시험 전 꼭 보기

Router(config)# exit

Router# copy r s

09. Router1 Telnet에 접근하는 Password를 “1234"로 설정하고, 상태를 저장하시오#

Router> en

Router# conf t

Router(config)# line vty 0 4

Router(config-line)# password 1234

Router(config-line)# login

Router(config-line)# exit

Router(config)# exit

Router# copy r s

10. Telnet에 접속 후 5분 50초 동안 신호가 없을 시 해당 세션을 자동으로 종료하도록 설정하시오#

Router> en

Router# conf t

Router(config)# line vty 0 4

Router(config-line)# exec-timeout 05 50

Router(config-line)# exit

Router(config)# exit

Router# copy r s

11. console에 5분 50초 동안 신호가 없을 시 해당 세션을 자동으로 종료하도록 설정하시오#

Router> en

Router# conf t

Router(config)# line con 0

Router(config-line)# exec-timeout 05 50

Router(config-line)# exit

Router(config)# exit
Router# copy r s

12. Router1의 console 패스워드를 1234로 설정하고, 저장하시오#

Router> en

Router# conf t

Router(config)# line con 0

Router(config-line)# password 1234

Router(config-line)# login

Router(config-line)# exit

Router(config)# exit

Router# copy r s

13. Router2의 interface Serial 0/0을 활성화시키고 저장하시오#

Router> en

Router# conf t

Router(config)# int se0/0

Router(config-if)# no shut

Router(config-if)# exit

Router(config)# exit

Router# copy r s

14. enable password를 1234로 설정하고 저장하시오#

Router> en

Router# conf t

Router(config)# enable password 1234    //모든 설정은 Router(config)# 에서 가능

Router(config)# exit

Router# copy r s

15. 패스워드를 1234로 설정하고 암호화 하시오#

Router> en

Router# conf t

Router(config)# enable secret 1234

Router(config)# exit

Router# copy r s

16. Hostname을 R1으로 변경하고, console 0의 패스워드를 1234로 변경하시오.#

Router> en

Router# conf t

Router(config)# hostname R1

R1(config)# line con 0

R1(config-line)# password 1234

R1(config-line)# login

R1(config-line)# exit

R1(config)# exit

R1# copy r s

17. Router1 DHCP 네트워크를 192.168.100.0/24, 서버 이름은 “icqa"로 설정하시오#

Router> en

Router# conf t

Router(config)# ip dhcp pool icqa   // 시험 전 꼭 보기

Router(config)# network 192.168.100.0 255.255.255.0 // 시험 전 꼭 보기

Router(config)# exit

Router# copy r s

18. R1에 access-list 1이 설정되어 있을 때 FastEthernet 0/0에 적용하시오#

R1> en

R1# conf t

R1(config)# int fa0/0

R1(config-if)# ip access-group 1 in // 시험 전 꼭 보기

R1(config-if)# ip access-group 1 out    // 시험 전 꼭 보기

R1(config-if)# exit

R1(config)# exit

R1# copy r s

19. domain-name을 “ICQA"로 설정하고 저장하시오#

R1> en

R1# conf t

R1(config)# ip domain-name ICQA // 시험 전 꼭 보기

R1(config)# exit

R1# copy r s

20. R1이 R2를 통하여 인터넷 사용이 가능하도록 정적 라우팅을 설정하시오#

R1> en      // R1이 R2를 통해서 : R1에서 세팅

R1# conf t

R1(config)# ip route 192.168.20.0 255.255.255.0 172.16.1.2

// 처음 상대방 IP는 네트워크 주소만 입력(마지막은 0)
// R1의 IP 정보는 사실상 필요 X

R1(config)# exit

R1# copy r s

09. 서비스 관리자#

1. 날짜 및 시간 동기화의 유지 관리를 자동으로 수행 (Windows Time)

2. ISCSI Initiator 서비스 (Microsoft iSCSI Initiator Service)

3. IP 주소를 할당 받는 클라이언트가 IP 주소를 자동으로 할당 받지 못하고 있다 ~ (DHCP Server)

4. 이 서비스는 암호화된 파일을 NTFS시스템 볼륨에 저장하는데 ~ (Encryption File System)

5. Windows와 기타 프로그램의 업데이트를 검색하고 ~ (Windows Update)

6. 인터넷 정보 서비스 관리자를 사용하여 ~ (World Wide Web ~)

10. Linux#

Run Level 0 : 종료

Run level 1 : 싱글 모드

Run level 5 : Xwindow 모드

Run level 6 : 재부팅

Inetd 방식

  • 응답속도가 느림
  • 메모리 사용량이 적음
  • Telnet, FTP 서버로 사용 할 때 많이 사용

인터넷 계층 : RARP, ARP, IP, ICMP, IGMP

전송 계층 : TCP, UDP

echo 0 : ping 정상 작동

PLC : 전력선에 데이터를 얹어 보내주는 기술

유비 쿼터스 : 언제 어디서나

AP : 공유기

SNMP : 관리(Manager) + Protocol

렉(Rack) : 장비 수용 철체 프레임

IPSec : 가상전용 회선을 구축하여 데이터 도청 방지 통신 규약

DMZ : 안전한 통신을 실현하는 통신 규약

FTP : 20, 21번

Telnet : 23번

SSH : Telnet의 진화 버전 (22번)

SSL : http - https

게이트웨이 : 2개 이상의 통신망을 연결시켜주는 장치

브릿지 : 데이터링크 계층 간의 패킷 전송을 담당하는 장치

OUI : MAC 주소의 앞쪽 번호

Hop count : 라우터 개수

VPN : 기존의 인터넷 망을 그대로 이용하면서 전용선을 이용하는 효과가 있는 것

UDP : 비신뢰성, 비연결형, 순서형

CSMA/CA : 충돌 회피(무선)

FSK : 주파수를 이용한 ~

OFDM : 주파수 성분이 겹쳐 있어도 ~

MAU : Ethernet AUI Port 선택하셈
